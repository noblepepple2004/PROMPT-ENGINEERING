# AI Response Evaluation Prompt

## Objective

Evaluate the quality of an AI-generated response.

## Prompt

Review the AI response using the following criteria:

- Accuracy
- Completeness
- Clarity
- Logical reasoning
- Grammar
- Helpfulness

Provide:

1. A score out of 10.
2. Strengths.
3. Weaknesses.
4. Specific suggestions for improvement.

---

## Why This Prompt Works

This prompt encourages structured evaluation rather than subjective opinions. It mirrors the type of work many AI trainers and evaluators perform.
